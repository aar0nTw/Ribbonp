Ribbonp

A Octopress plugin to output p block with pure css ribbon.

How to Use

{%ribbonp (info|warning) <custom title>%}
Woooo Ribbon P
{%endribbonp%}

Select your style like {%ribbonp info%} will output the blue color style's p block w/ ribbon.
